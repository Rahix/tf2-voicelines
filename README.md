tf2-voicelines
==============

tf2-voicelines aims to be an easy way to find quotes of Team Fortress 2 characters.

Built using [Vue](https://vuejs.org/), [Lodash](https://lodash.com/) and [Flat-UI](https://github.com/designmodo/Flat-UI).
